# ClimaShock API

Pakistan's first distributed causal climate-economic intelligence system.

## Endpoints
- `GET /` — system info
- `GET /health` — health check
- `POST /predict` — cascade prediction
- `GET /causal` — causal links
- `GET /districts` — district info
- `GET /discoveries` — novel findings
- `GET /docs` — interactive API docs
